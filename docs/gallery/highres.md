# As in thumbnail

## In OccuPy

1. Fetch EMDB 33707
2. Set scale mode = occupancy
3. Estimate scale 
4. Launch ChimeraX (open chimX_emd_33707.cxc through ChimeraX)

## In chimeraX 

```commandline
vol gaussian #1 sdev 0.8 
scale_color #5 #2
```
